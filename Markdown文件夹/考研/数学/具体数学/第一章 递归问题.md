# 递归问题

## HANOI（汉诺塔）

> 命名并求解：其实就是定义状态，比如$T_n$表示有$n$个塔时的最少移动次数。不管如何移动盘子，都要先把$n-1$的塔挪到第二个塔，最下的拿到第三个塔，再将$n-1$的塔挪到第三个塔。根据上面的流程，易得$T_n = 2T_n + 1$。（严格来说是确定两个不等式分别对应上下界，利用夹逼法则来确定等式）

> 数学归纳法：基础*basis*证明相对于问题的$n_0$成立。假设$0\sim n-1$时成立。最后通过推关于$n$的式子，推到式子为假设的式子，此时对$n$成立。这步称作归纳*induction*。

> 递归的封闭形式：与通项式意义相同。

## 平面上的直线

>问题是一个平面上放置$n$条直线（两两互不平行）会划分成多少个区域。
>
>可以看到直线经过哪些区域，就会划分哪些区域，且直线在一个被经过的区域只会出现连续的一段线。即有$T_n = T_{n-1} + n$。易给出封闭形式。

> 对于一些重要的 **数** ，比如$1*2*\dots*n$，虽然并非是封闭形式，但定义了$n!$，其为封闭形式。即封闭形式不局限于式子仅含四则运算和幂指运算。

> 对于折线来说，如果要求最多划分数，需将每条折线的锯齿点放在交点“之外”，此时划分区域最多。而对于每个折线来说，反向拓展射线，可以发现每个折线会多出两个区域。故$R_n = T_{2n} - 2n$

## 约瑟夫问题

>$n$个人，初始从第二个人开始踢，每隔一个位踢一个人，问最后剩编号几。
>
>考虑$2n$的问题：第一轮踢完偶数编号的人，发现问题转化为规模为$\frac{n}{2}$，对应编号为$2i-1$。故$J(2n) = 2J(n) - 1$。
>
>考虑$2n+1$的问题：第一轮踢完偶数编号加上编号1，发现问题转化为规模$\frac{n-1}{2}$，对应编号为$2i+1$。故$J(2n+1) = 2J(n) + 1$。
>
>初始有$J(1) = 1$。
>
>给出表格，发现有$n = 2^k + l,l<2^k,J(n) = 2l + 1$。对$k$进行归纳法可证。
