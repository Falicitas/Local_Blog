# 第二章 合式

## 记号

> 序列一般写三项足矣。比如：$2^{0}+2^{1}+\cdots+2^{n-1}$

> $\sum$形式有两种：一是有确定上下界的形式：
>
> $\sum_{k=0}^{49}(2 k+1)^{2}$
>
> 一种是把条件放在求和符号下：
>
> $\sum\limits_{p \leq N \\ p是素数} \frac{1}{p}$
>
> 第二种更方便对指标做变换，比如$\sum_{1 \leq k \leqslant n} a_{k}=\sum_{1 \leqslant k+1 \leqslant n} a_{k+1}$，第一种通常在写结论时使用。

> 艾佛森约定：用中括号来表示命题真假的约定。比如$[p \text { 是素数 }]=\left\{\begin{array}{l}
> 1, p \text { 是素数 } ; \\
> 0, p \text { 不是素数. }
> \end{array}\right.$
