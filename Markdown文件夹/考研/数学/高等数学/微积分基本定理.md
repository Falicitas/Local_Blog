# 微积分基本定理

## 反导函数

即函数$f$的**不定积分**。带常数项$c$。在一段连续的定义区间内，反导函数之间属于向上向下平移的关系，即$(a,b)$内导数处处相等。

对于在不同的定义区间，反导函数的常数项可以不同，比如

$F(x)=\left\{\begin{array}{ll}-\frac{1}{x}+C_{1} & x<0 \\ -\frac{1}{x}+C_{2} & x>0\end{array}\right.$

是$f(x) = \frac{1}{x^2}$的不定积分，$C_1,C_2$可以不相等。

另外，反导函数也可由微积分基本定理得来：$F$为$f$的反导函数，$F(a) = 0$，则$F(x) = \int_{a}^x f(t)dt$。若$F(a) = A$，$F(x) = F(x) = \int_{a}^x f(t)dt + A$，即$f$可根据反导函数的**一个特殊值和积分**来唯一确定一个反导函数。

## 微积分基本定理

如果函数 $f$ 是闭区间 $[a, b]$ 上的连续函数, $F$ 是 $f$ 在 $[a, b]$ 上的一个反导函数, 那么有

$\int_{a}^{b} f(x) \mathrm{d} x=F(b)-F(a)$

证明：

现有$[a,b]$的分割$P$，即有限点列$a=x_0<x_1<...<x_n=b$。

对于每个子区间$[x_i,x_{i+1}]$，根据柯西中值定理定理，有$F(x_{i+1})-F(x_i) = f(\xi_i)\Delta x_i,\xi_i\in(x_i,x_{i+1})$

将所有子区间的等式累加在一起，得$F(b) - F(a) = \sum\limits_{i=0}^{n-1}f(\xi_i)\Delta x_i$

由于$f$在$[a,b]$上连续，故其黎曼可积，利用黎曼积分的定义可得$F(b)-F(a) = \int_{a}^b f(x)dx$。
