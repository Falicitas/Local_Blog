# 高斯积分 & $\Gamma(x)$函数

虽然不定积分$\int e^{-x^{2}} d x$不能用初等函数表示，但定积分$\int_{-\infty}^{\infty} e^{-x^{2}}$是可以计算的。

结果为$\int_{-\infty}^{\infty} e^{-x^{2}} = \sqrt{\pi}$

任意的高斯积分：$\int_{-\infty}^{\infty} e^{-a(x+b)^{2}} d x=\sqrt{\frac{\pi}{a}}$

## $\Gamma(x)$函数

定义$\Gamma(x) = \int_0^{\infty} t^{x-1}e^{-t}dt$。

对于正整数$n$来说，$n! = \Gamma(n+1)$。

证明：

对$\Gamma(x+1)$使用分部积分，$t^{x}$为求导项，$e^{-t}$为积分项，有：

$\Gamma(x+1) = \int_0^{\infty} t^{x}e^{-t}dt = -t^xe^{-t}|_0^{\infty} - x\int_0^{\infty}t^{x-1}e^{-t}dt = x\Gamma(x)$

$\Gamma(1)=\int_{0}^{+\infty} \mathrm{e}^{-t} \mathrm{d} t=0-(-1)=1 = 0!$

证毕。

## 高斯积分与$\Gamma(x)$的关系

$\Gamma(x) = \int_0^{\infty} t^{x-1}e^{-t}dt$

$\Gamma(\frac{1}{2}) = \int_0^{\infty} t^{-\frac{1}{2}}e^{-t}dt$，换元$x^2 = t,dt = 2xdx$

有$\Gamma(\frac{1}{2}) = 2\int_0^{\infty}e^{-x^2}dx = \int_{-\infty}^{+\infty}e^{-x^2}dx = \sqrt{\pi}$

所以以半整数Gamma函数都带$\sqrt{\pi}$。
