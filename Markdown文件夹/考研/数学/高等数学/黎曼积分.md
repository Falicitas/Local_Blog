# 黎曼积分

黎曼积分指给区间上的积分下一个明确的定义，即可理解为二维上的区间定积分皆可称作黎曼积分。

其可用来推导微积分基本定理。

## 黎曼和

设区间$[a,b]$的**分割P** = 有限的点列$a = x_0 < x_1 < ... < x_n = b$，定义$t_i$为在$[x_i,x_{i+1}]$上所取的函数值，则有**黎曼和**$\sum\limits_{i=0}^{n-1}t_i(x_{i+1}-x_i)$。

黎曼积分可由$\epsilon-\delta$定义：$S$为函数$f$在$[a,b]$上的黎曼积分，则对于任意$\epsilon > 0$，都存在$\delta>0$，当对应的分割$P$的最长区间$\lambda < \delta $时有$|\sum\limits_{i=0}^{n-1}t_i(x_{i+1}-x_i) - S| < \epsilon $。

## 黎曼可积

函数在闭区间$[a,b]$上连续，则黎曼可积。
