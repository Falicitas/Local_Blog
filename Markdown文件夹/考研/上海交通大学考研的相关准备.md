# 上海交通大学考研的相关准备

不一定去清华or上交的。在备考时要充分了解读研要考虑的各类情况，比如导师，地理位置，科研环境，etc。

##考试内容

初试考试科目编码及名称：
> * ① 101 思想政治理论	
> * ② 201 英语一
> * ③ 301 数学一	
> * ④ 408 计算机学科专业基础综合

###各科考试内容

####101 思想政治理论 （分值：100）
> * 马克思主义基本原理概论 约24 %
> * 毛泽东思想和中国特色社会主义理论体系概论 约30 %
> * 中国近现代史纲要 约14 %
> * 思想道德修养与法律基础 约16 %
> * 形势与政策以及当代世界经济与政治 约16 %

预选参考书：
1、命题人知识点提要 肖秀荣
2、肖四（肖秀荣考前四套体）
***or***
1、《全国硕士研究生入学统一考试-思想政治理论考研大纲解析》俗称红宝书
2、《考研政治大纲解析配套习题同步训练1600题》是与红宝书配套使用的

##分数线问题

初试考试是去参加统一出题试卷的，类似于高考。

初试成绩后，院校会采取两种分数线制：一种是国家线，一种是自主线。我只需看上海交通大学的自主分数线。

##时间表
> * 大一
    周一到周五跟董世龙或者战折每日约图书馆，在完成学校功课的前提下，花最多时间（不牺牲陪老婆的时间）在acm中。大一将英语六级攻下。
> * 大二
    大二开始联系上交的招生官，了解多一些信息，专业课一定要学好。大二上学期准备雅思。
> * 大三
    大三上学期了解上交的更多信息，看能否结交一些上交的学长。雅思冲击7.0以上。大三下学期就开始考研准备。
> * 大四
    大四继续学习8，编程和英语双发展。
