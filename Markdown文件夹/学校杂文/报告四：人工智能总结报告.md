在第四次信息技术导论课上，教授为我们简要介绍了人工智能方面的应用与实现。在本次报告中我收获了不少，了解到机器如何学习知识与进行推理，并在课下阅读了不少相关文章与文献。此次总结报告将分为人工智能与人类智能的联系与人工智能的应用领域两方面进行总结。

##人工智能与人类智能
对于目前人工智能领域的研究而言，领域内的专家们尚未能准确浓缩定义人工智能。这个其实挺难定义的。在人工智能大会（AAAI 2018）的审稿人系统里提供了几个子领域选项（笔者在此不一一列举。共计25项子领域）：
> * AI and the Web
> * Cognitive Modeling
> * Cognitive Systems
> * Computational Sustainability and AI
> * Game Playing and Interactive Entertainment
> * Game Theory and Economic Paradigms
...

但归结来说，人工智能是研究如何让人类制造的智能体（程序，机器人等）模拟人类智能或者在一些方面超越人类智能的一类科学。因此了解人类智能的组成与运作机制，对理解人工智能方能有更好的帮助。
根据人类心智进化的历程，人类心智从初级到高级可以分为五个层级: 神经层级的心智、心理层级的心智、语言层级的心智、思维层级的心 智和文化层级的心智[^thesis]。在目前人工智能领域的发展来看，人工智能已经踏入了神经、心理、语言、思维四大领域并有了大小不一的进展。尤其是神经层级与思维层级的发展，创就了诸如AlphaGo与深蓝在棋类这样原属于人类智能结晶领域的成就，并掀起了人们对人工智能的关注。


![p1](http://image.cnki.net/tempfile/BDZK2016040160002_w.jpg)


笔者仅从人工智能模仿人类智能的神经领域上进行浅显探讨。在人工智能的神经网络领域上，人工智能参照着人类智能，以一种机器化的思维机制呈现在人们的视野。神经网络的信息处理是由神经元之间的相互作用实现的：知识与信息的存贮主要表现为网络元件互连间分布式的物理联系。人工神经网络具有很强的自学习能力，它可以不依赖于“专家”的头脑，而自动从已有的实验数据中总结规律。由此，人工神经网络擅长于处理复杂多维的非线性问题，不但可以解决定性问题，也可解决定量的问题，同时还具有大规模并行处理和分布的信息存储能力，具有良好的自适应、自组织性以及很强的学习、联想、容错和较好的可靠性。总结来说，人工智能在神经网络领域上的运转机制参照于人类智能，而且有望赶上人类智能，甚至超越它。

##人工智能的应用领域
笔者在阅读完相关文献后，发现人工智能的应用领域远远不只是社会上广为讨论的诸如AlphaGo人工智能机器人，Sophia女性机器人，微软小冰等热门的机器人智能那么简单，其更多能用于帮助解决很多迫在眉睫的社会问题和社会生活中的重要挑战。
在此举一个在人工智能领域颇有建树的方飞博士的研究方向及实际解决的社会问题：

Game Theory and Economic Paradigms（博弈论与经济范式）。设计算法来计算在涉及连续时空的安全博弈问题中的均衡策略，这个算法也被应用到设计巡逻船路线，来保护纽约港口的史丹顿岛轮渡。

Machine Learning Applications（机器学习应用）。比如通过非常稀疏而且有不确定性的数据来预测乌干达的保护区内的捕猎夹分布，从无人机拍摄的热红外影像视频中自动识别野生动物偷猎者等。

Computational Sustainability and AI（计算性可持续发展与人工智能）。基于机器学习和博弈论的算法，在马来西亚的保护区内设计护林员巡逻路线。[^article]

---

##总结

由此可见，人工智能的运用范围广泛且有深度，已经对人类社会生活与活动创造的丰富的社会价值。人工智能相关领域的研究成果已被广泛地应用于国民生活、工业生产、国防建设等各个领域。在信息网络和知识经济时代，人工智能技术正受到越来越广泛的重视，必将为推动科技进步和产业的发展发挥更大的作用。
