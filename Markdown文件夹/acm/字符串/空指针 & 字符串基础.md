# 空指针 & 字符串基础

仅记录用的不熟的概念与函数。

## 常见概念

- 字符集

一个 **字符集** $\Sigma$ 是一个建立了全序关系的 **集合** , 也就是说, $\Sigma$ 中的任意两个不同的元素 $\alpha$ 和 $\beta$ 都 可以比较大小, 要么 $\alpha<\beta,$ 要么 $\beta<\alpha$。 字符集 $\Sigma$ 中的元素称为字符。

- 字符串字面值

即 字符串常量 。对应的有 字符字面值 ，即 字符常量 。

## 常用标准库函数

- sscanf

`sscanf(const char *__source, const char *__format, ...)`，从字符串里读入对应格式的信息。

比如`sscanf(str,"%d",&a)`。

- sprintf

`sprintf(char *__stream, const char *__format, ...)`，从字符串里输出对应格式的信息。

比如`sprintf(str,"%d",i)`。

- strcmp

`int strcmp(const char *str1, const char *str2)`，str1字典序小返回负值，大返回正值，否则为0。

**注意不同平台不一定返回的都是-1,0,1，但都返回负值，0，正值**。

- strncpy

`char *strncpy(char *str, const char *src, int cnt)`，写入至多cnt个字符到str里。src不足cnt有效字符则会补`\0`至cnt个。

- strcat

`char *strcat(char *str1, const char *str2)`，将str2接到str1的末尾。

- strstr

`char *strstr(char *str1, const char *str2)`，找到str1里第一个出现的str2的首个位置，否则返回NULL。

- strrchr

`char *strrchr(const char *str, char c)`，找到在字符串 str 中最后一次出现字符 c 的位置，并返回这个位置的地址。如果未找到该字符则返回 NULL 。没有`strrstr`

## stl的string

- 赋值运算符 `=` 右侧可以是 `const string/string/const char*/char*` 。
- 访问运算符 `[cur]` 返回 `cur` 位置的引用。
- 访问函数 `data()/c_str()` 返回一个 `const char*` 指针，内容与该 `string` 相同。
- 容量函数 `size()` 返回字符串字符个数。
- 还有一些其他的函数如 `find()` 找到并返回某字符位置。
- `std :: string` 重载了比较逻辑运算符，复杂度是 $O(n)$ 的。
- `substr(pos,len)`返回从pos开始，长度为len的string类型子串。

## 空指针常量

> NULL是宏，在某头文件里有`#define NULL 0 `（还有另一种形式`(void*)0`），平常就当做0即可。
>
> nullptr是空指针常量。由于NULL既算空指针常量又算整型常量，会引起二义性，故c++里用nullptr给指针赋值，将其当做函数参数就不会产生“两个同名函数，一个参数为int一个参数为某类型指针”的调用错误问题。
