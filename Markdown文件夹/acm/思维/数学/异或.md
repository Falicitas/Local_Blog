# 异或

## 一些习题

### CF1438D Powerful Ksenia

先对区间内三个元素取异或，再赋值给三个元素。

异或的一个性质，`a ^ a = 0,0 ^ a = a `。根据这个性质，可以让两个相同的数$a_i,a_j$变成另一个数$a_k$。

于是可以这样操作：下标1,2,3异或得到三个相同数，3,4,5异或得到三个相同数，...，最后三个（n为奇数）或者3个+1个（n为偶数）。此时(1,2),(3,4),...(2n-1,2n)的数都是相同的。

如果n为奇数，只要让每个对和下标n取个异或就可以得到全部为n的元素；

n如果是偶数？令$T = a_1 \oplus a_2 \oplus \dots \oplus a_n$，可以证得每次操作不改变区间异或值：每次操作相当于对$T\oplus a_i \oplus a_j \oplus a_i \oplus a_k \oplus a_j \oplus a_k = T$。那么当像奇数那样操作后，势必$a_{n-1} = a_n$，当总体异或为0（由于$a_{n-1} = $前面n-1个元素的异或和）。故得出结论，当n为偶数时，只要整体异或为0，就存在答案。反之不存在答案。
