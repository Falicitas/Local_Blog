# 打怪题

## 一些习题

### CF1435E Solo mid Oracle

在初始0时刻，给敌方英雄$a$点瞬时攻击。敌方英雄在受攻击后$1,2,\dots,c$每秒$+b$血。每次发动技能的间隔为$d$。问在哪一秒打出的总伤害（积累伤害）最高。若无解输出-1（即永远有更大的伤害）。回血能叠加。

将时间推至无限，那有解的情况必定是$a\leq bc$。

一般会先从贪心打怪的思路想起。在这题别着急，先按 怪受攻击+怪完全恢复 这个步骤作为一个循环单位进行分析。由于$a\leq bc$，当在$c$时刻，第一次的攻击的贡献就已经是非正数了，由此分析可确定最佳时刻必定在$[0,t],$其中$t$为$a-bt>0$的最大$t$。利用取整公式得到$t = \left\lceil{\frac{a}{b}}\right\rceil - 1$。由于在$[0,t]$中，打出的任何伤害在此区间都是正贡献，故尽量多打，易推出共打$\frac{t}{d} + 1$次。最后再计算回复的血整合到一起就行了。
