# 优先队列

## 声明

```cpp
struct cmp1{
    bool operator ()(int &a,int &b){
        return a<b;//最大值优先
    }
};

struct cmp2{
    bool operator ()(int &a,int &b){
        return a>b;//最小值优先
    }
};

priority_queue<int,vector<int>,cmp1> pq;//最大值优先
priority_queue<int,vector<int>,less<int> > pq;//最大值优先
```
